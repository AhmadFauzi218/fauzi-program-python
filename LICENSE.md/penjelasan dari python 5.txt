str (string)

nama  = "ahmad"

-menempatkan lokasi memory tersebut dengan identifier NAMA
-karena nilai nya karakter ,maka tipe varabel ini adalah STR

IDENTIFIER nama DENGAN NILAINYA "ahmad" TIPE str

python
